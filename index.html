전세계가 편리해지는 서비스를 구축하는데 앞장서겠습니다.기술과문화를 연결해 온 나노아이티가 세상의 디지털 라이프 스타일을

Fun SMS

자체 개발한 솔루션 [웹서비스-웹사이트, PC 메신저] [Mto 모듈-DB 연동방식, API 연동방식]으로 통합 메시지
[SMS, LMS, MMS / 카카오 메시지-알림톡,친구톡] 발송 시스템을 제공 합니다.) (통합 메시지 시스템/ 비즈
메시지 서비스<문자메시지 / RCS / 웹서비스> / 카카오 메시지 < 알림 톡 / 친구 톡 >)

Anti - DDos

GEO DDNS를 이용하여 ISP(인터넷 회선 공급자사)별 분산 방어시스템으로 120G 이상의 대용량 대역폭 공격에도
고객의 서비스연속성을 보장하는 국내 유일의 특허 받은 디도스 우회방어 서비스 Advanced Security 500G 인터넷
연동으로 디도스 공격을 방어 합니다.

My VPN

myVPN은 PPTP Tunneling 기술을 이용하여 ADSL, VDSL, 광랜 등 초고속 인터넷 회선에
고정 IP (공인IP)를 제공하는 서비스입니다.

o2o Service

마트(슈퍼마켓)시장에 스마트워크 플랫폼을 구축하여 새로운 전국 유통채널을 창조합니다.
마트의 Needs에 부합한 쇼핑몰을 제공하고 상품이미지DB를 구축뿐만 아니라 배달중계 서비스로
마트의 운영을 도울 수 있는 서비스 를 제공합니다.

핵심가치

Core Values

나노아이티의 핵심 가치는 ‘고객과의 동반
성장’입니다. 고객과 기업, 사람과 사람을
연결하는 서비스를 제공하는 기업으로서,
언제나 고객 입장에서 생각하고 고객의 성장을 지원합니다. 고객 성장이 곧 저희의 성장 이라는 가치 아래 내부 고객 성장도 중요하게 여기며, 지속가능한 성장을 위해 ‘이음’과 ‘도움’ 을 주는 서비스 제공을 목표로 합니다.

기술을 하나의 ‘아트(ART)’라고 본다면, 저희는 그것을 잇는 역할을 하고 싶다는 의지를 담아 사옥 이름도 ‘ARTBRIDGE’라고 지었습니다. 대구 디지털 산업의 중심인 수성 알파시티에서 교육/세미나 공간과 공유 오피스를 운영하는 것도 다양한 기업이 함께 성장할 수 있도록 돕는 ‘가교’ 역할을 하고자 하는 바람이 담겨있습니다.

2005년 EBI(E-Business Integrator) 기업으로 출발,'DDOS 난공불락'을 선보이며 디도스 방어의 대중화에 앞장 서 온 나노아이티는사람과 세상을 잇고 미래가 필요로 하는 IT 서비스를 만들기 위해 매진해왔습니다.
